网页版 新T树洞：[https://hole.thu.monster](https://hole.thu.monster)

## 获取源码

`git clone --recurse-submodules https://git.thu.monster/newthuhole/hole_thu_frontend.git`

注意,由于改造自P大树洞网页版额，包含了由xmcp维护的submodule

## 部署

推荐使用yarn

+ 本地调试
```
yarn install
yarn start
```

+ 打包部署
```
yarn run build
```

## 后端接口

+ /\_login
+ /\_api/v1/...
